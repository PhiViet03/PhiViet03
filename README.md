# 💫 About Me:
Hi, I’m Phi — a fresh graduate in IoT and Applied AI who builds real stuff instead of just talking about it. I work mainly with Python, especially FastAPI for backend development, and I’ve deployed actual APIs using Docker and cloud platforms, so I understand how things run outside of my laptop (and how they break too). I also have a strong interest in AI and computer vision, working with tools like TensorFlow, MediaPipe, and OpenCV to build models that actually do something useful in real time. On the hardware side, I’ve worked with LiDAR, ROS2, and embedded systems, so I’m comfortable dealing with both code and physical devices when needed. I’d say I’m the type who learns fast, figures things out under pressure, and doesn’t give up until something works — even if it takes way too long. Right now, I’m looking for an opportunity to gain real-world experience, contribute to actual products, and grow into a solid engineer… and yeah, also because I need a job 😄.<br>
